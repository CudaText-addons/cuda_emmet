Plugin for CudaText.
Allows to use basic commands of the Emmet web-developers engine.
The engine itself is built-in in the CudaText, and it's written
in Pascal by Rickard Johansson, https://www.rj-texted.se/ .

Includes file "help.html" with the Emmet cheat-sheet.
